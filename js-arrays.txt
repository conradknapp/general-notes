How do you use .slice()?

.slice() removes an element(s) of a given array and return a copy (doesn't mutate the original)

var people = ["Conrad", "Brad", "Steve"];

var newPeople = people.slice(2);

people; // ["Conrad", "Brad", "Steve"]

newPeople; // ["Steve"]
